HTML

HyperText Markup Language

HTML is a web building piece that defines the structure of web content. It's used to make web pages and applications. It is the coding language that the web uses to create various HTML elements. It can be assisted by technologies such as CSS and JavaScript. HTML is the language that computers use to interact with one another on the internet.

CSS

Cascading Style Sheets

CSS or Cascading Style Sheets is is a straightforward way for adding style (for example, fonts, colors, and spacing) to Web documents. It is a computer language for laying out and structuring web pages or to show how a website will look like. This is commonly used to make a website look much better. In simpler words, CSS is a styling language that allows you to style your webpage in an efficient manner.

Bootstrap

Bootstrap Version 4

Bootstrap is a framework that helps in designing websites faster and easier. It gives us the ability to easily design our websites. There are four reasons why a lot of people use Bootstrap- one, it is very easy to use, two, it has very responsive features, three, it believes in a mobile-first approach and lastly, it is very compatible with browsers.
